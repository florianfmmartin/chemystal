# chemystal

A reflexive chemical abstract machine (RCHAM) based concurrency library inspired by join-calculus

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     chemystal:
       github: florianfmmartin/chemystal
   ```

2. Run `shards install`

## Usage

```crystal
require "chemystal"
```
